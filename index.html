📖 Glossary / 用語集

日本語English説明 / Description
散布図Scatterplot2つの量的変数の関係を点で視覚化したグラフ。各点は1観測値を表す。
方向Direction散布図で右に行くと上がる（正）か下がる（負）か、または傾向なしかを示す。
形状Form散布図のパターンが直線的か非線形（曲線）かを示す特性。
強さStrength点が全体的なトレンドにどれだけ密集しているかの度合い。
変わった特徴Unusual Features外れ値・クラスター・ギャップなど、全体パターンから外れる特徴。
正の相関Positive Association一方が増えるともう一方も増える傾向。
負の相関Negative Association一方が増えるともう一方が減る傾向。
相関係数Correlation Coefficient (r)2つの量的変数の線形関係の向きと強さを−1から1で表す指標。
Pearson rPearson Correlation Coefficient標準化された共分散として定義される、最も一般的な相関係数。
共分散Covariance2変数がどれだけ一緒に変動するかを測る指標。単位はx×yの積になる。
外れ値Outlier全体のパターンから大きく外れた観測値。r に大きな影響を与える。
相関行列Correlation Matrix複数の量的変数すべてのペアの相関係数を表にまとめたもの。
散布図行列Scatterplot Matrix複数変数のすべてのペアの散布図を格子状に並べたもの。
対称行列Symmetric Matrix対角線を軸に左右が等しい行列。相関行列はcor(A,B)=cor(B,A)のため対称。
多重比較Multiple Comparisons複数のペアを一度に比較すること。type I error が増加しやすい。
自由度Degrees of Freedom (df)統計的に「自由に変動できる」値の数。標本分散の計算ではn−1を使う。
標準化Standardization変数を平均0・標準偏差1に変換すること（Zスコア化）。
ZスコアZ-score観測値が平均から何標準偏差離れているかを示す値：z=(x−x̄)/s。
線形変換Linear Transformation定数を掛けるか足す変換（y=ax+b）。r は変化しない。
単位なしUnitlessrには測定単位がない。純粋な数値。
不抵抗Not Resistant外れ値の影響を受けやすい性質。rはnot resistantである。
相関≠因果Correlation ≠ Causation相関関係があっても因果関係とは言えない。第三の変数（潜伏変数）が存在する可能性。
潜伏変数Lurking Variable測定されていないが、X・Y両方に影響する第三の変数。
逆因果Reverse CausationX→Yに見えるが、実はY→Xである可能性。観察データでは方向性が不明。
Spearman相関Spearman Correlation順位ベースの相関。外れ値に対してPearson rより頑健。
EDAExploratory Data Analysisデータを正式な分析の前に探索的に調べる作業。可視化・外れ値・欠損値の確認を含む。
探索的データ分析Exploratory Data Analysisモデリング前にデータの構造・質・パターンを理解する分析段階。
単調な関係Monotone Relationship一方向にのみ増加または減少する関係。Spearman rが有効。
完全な線形関係Perfect Linear Relationshipr=1またはr=−1で、全点が完全に一直線上に並ぶ状態。
r≈0の落とし穴Pitfall of r≈0r≈0は線形関係がないことを示すが、非線形関係（曲線など）は存在しうる。
